Google Sheets Integration
View Step-by-Step Setup Guide
  1. Open your Google Sheet:
    Spreadsheet link.
  2. Click **Extensions** -> **Apps Script**.
  3. Create a script by copy-pasting the contents of `google-apps-script.js` from the Wrext repository.
  4. (Optional) Add a script property `API_TOKEN` in project settings for security.
  5. Click **Deploy** -> **New deployment**.
  6. Set type to **Web App**, set *Execute as* to **Me**, and set *Who has access* to **Anyone**.
  7. Click **Deploy**, authorize permissions, copy the Web App URL, and paste it into the field above.
